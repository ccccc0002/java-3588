## 平台概述  
AI视频管理平台提供专业、高效的实时视频监控解决方案，致力于消除芯片厂商间的技术壁垒，实现GPU与NPU芯片的统一调度与管理。平台无缝兼容现有监控基础设施，深度融合智能分析算法，构建从芯片层到应用层的端到端协同体系。<br>通过标准化接口与模块化设计，企业可显著降低开发成本（最高达95%），并借助直观的图形化界面快速完成视频源接入、智能布控及策略配置。<br>该平台不仅强化业务安全性，更以高效、灵活、经济的运营模式，助力企业实现智能化升级与可持续增长。

## 核心优势  
0. **合作模式多样**：自研代码，支持任意形式合作，自带LOGO替换改名功能，支持**二次功能定制化开发**。
1. **一体化平台**：集成视频监控、推理计算、告警通知及数据标注功能，提供端到端解决方案。
2. **协议兼容**：全面支持**GB28181/RTSP/RTMP**协议，兼容**H.265/H.264/视频编码。  
3. **跨架构部署**：支持**x86/ARM**指令集，适配多样化硬件环境。
4. **灵活组网**：支持多级组网架构，满足不同规模监控场景需求。
5. **算法扩展**：支持自定义AI模型部署，实现多路视频流并发算法实时计算。
6. **多模态告警**：集成语音电话、飞书/企业微信/钉钉/APP通知及第三方API、现场音柱与LED屏告警通道。
7. **高并发处理**：支持多路视频流与多算法并行分析，保障告警响应时效性与系统稳定性。
8. **支持私有化**：支持项目私有化部署，按项目情况**支持源代码交付**。

####  **5.人流量统计** 

人流量统计模块用于监测和统计特定区域内的人流情况。可广泛应用于园区、办公楼、购物商场、博物馆、公共展会、车站等场景，能够为管理者提供重要的人流数据支持，有助于优化空间布局、提升服务质量和保障公共安全。
| 统计维度       | 技术定义                                                                 |
|----------------|--------------------------------------------------------------------------|
| **进入人数**   | 基于区域划分与统计线计算的进入人次                                       |
| **离开人数**   | 基于区域划分与统计线计算的离开人次                                       |
| **剩余人数**   | 同摄像机下进入人数与离开人数的差值（理论值可为负）                       |
| **统计数值**   | 全系统覆盖的计算单元与摄像机客流数据汇总（含进入/离开/剩余）             |
| **趋势分析**   | 按时间维度可视化展示总客流变化趋势（覆盖所有部署人流量统计算法的摄像机） |
| **单机统计**   | 按摄像机粒度细分客流数据，支持按时间范围动态展示变化趋势                |
